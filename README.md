# Flutter TIC-TAC-TOE Game
Looking for an interactive game? This game got you covered.

## Run Locally

Clone the project

```bash
  git clone https://github.com/yuvraj0028/tic_tac_toe.git
```

Go to the project directory

```bash
  cd tic_tac_toe
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
